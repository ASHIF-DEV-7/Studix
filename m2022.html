Find the inverse of the function f(x) = 3x + 2.

Answer:

Let y = 3x + 2
Solving for x: x = (y - 2)/3
Therefore, f⁻¹(x) = (x - 2)/3

Q12. 2 Marks

Evaluate: lim(x→2) (x² - 4)/(x - 2)

Answer:

lim(x→2) (x² - 4)/(x - 2) = lim(x→2) (x + 2)(x - 2)/(x - 2)
= lim(x→2) (x + 2) = 2 + 2 = 4

Q13. 2 Marks

If P(A) = 0.6 and P(B) = 0.4, find P(A ∪ B) if A and B are mutually exclusive events.

Answer:

Since A and B are mutually exclusive, P(A ∩ B) = 0
P(A ∪ B) = P(A) + P(B) - P(A ∩ B) = 0.6 + 0.4 - 0 = 1.0

Section C - Medium Answer Questions (4 marks each)
Q19. 4 Marks

Prove that √2 is an irrational number.

Answer:

Proof by Contradiction:
Assume √2 is rational, so √2 = p/q where p, q are integers with no common factors.
Then 2 = p²/q², so p² = 2q²
This means p² is even, so p is even. Let p = 2k.
Then (2k)² = 2q², so 4k² = 2q², so q² = 2k²
This means q² is even, so q is even.
But if both p and q are even, they have a common factor of 2, contradicting our assumption.
Therefore, √2 is irrational.

Q24. 4 Marks

Find the maximum and minimum values of f(x) = x³ - 3x² + 2 on the interval [0, 3].

Answer:

f'(x) = 3x² - 6x = 3x(x - 2)
Critical points: x = 0, x = 2
f(0) = 2, f(2) = -2, f(3) = 2
Maximum value = 2 (at x = 0 and x = 3)
Minimum value = -2 (at x = 2)

Q25. 4 Marks

Prove that the sum of angles in a triangle is 180°.

Answer:

Consider triangle ABC. Draw a line through A parallel to BC.
Let the angles be ∠BAC = α, ∠ABC = β, ∠BCA = γ
The parallel line creates alternate interior angles equal to β and γ
These angles plus α form a straight line = 180°
Therefore: α + β + γ = 180°

Section D - Long Answer Questions (6 marks each)
Q26. 6 Marks

A rectangular garden is to be fenced on three sides, with one side against a wall. If 100 meters of fencing is available, find the dimensions that maximize the area.

Answer:

Let width = x, length = y
Constraint: x + 2y = 100, so x = 100 - 2y
Area A = xy = (100 - 2y)y = 100y - 2y²
dA/dy = 100 - 4y = 0
y = 25 meters
x = 100 - 2(25) = 50 meters
Maximum area = 50 × 25 = 1250 square meters

Q29. 6 Marks

A manufacturer produces items at a cost of Rs. 50 each. The selling price is Rs. 80 each, but due to defects, 5% of items must be sold at Rs. 30 each. Find the expected profit per item and the probability distribution.

Answer:

Let X be the profit per item
P(good item) = 0.95, Profit = 80 - 50 = 30
P(defective item) = 0.05, Profit = 30 - 50 = -20
Probability Distribution:
X = 30 with P = 0.95
X = -20 with P = 0.05
Expected profit E(X) = 30(0.95) + (-20)(0.05)
= 28.5 - 1 = Rs. 27.5

Q31. 6 Marks

A company's profit P (in thousands) is related to the number of units sold x by P(x) = -2x² + 40x - 150. Find the number of units to be sold for maximum profit and the maximum profit.

Answer:

P(x) = -2x² + 40x - 150
For maximum, P'(x) = 0
P'(x) = -4x + 40 = 0
x = 10 units
Maximum profit P(10) = -2(100) + 40(10) - 150
= -200 + 400 - 150
= 50 thousand rupees
Since P''(x) = -4 < 0, this is indeed a maximum.

Q32. 6 Marks

Prove using mathematical induction that 1² + 2² + 3² + ... + n² = n(n+1)(2n+1)/6

Answer:

Base case: For n = 1: 1² = 1(2)(3)/6 = 1 ✓
Inductive hypothesis: Assume true for n = k
1² + 2² + ... + k² = k(k+1)(2k+1)/6
Inductive step: Prove for n = k+1
1² + 2² + ... + k² + (k+1)² = k(k+1)(2k+1)/6 + (k+1)²
= (k+1)[k(2k+1)/6 + (k+1)]
= (k+1)[k(2k+1) + 6(k+1)]/6
= (k+1)[2k² + k + 6k + 6]/6
= (k+1)(k+2)(2k+3)/6
= (k+1)((k+1)+1)(2(k+1)+1)/6 ✓

Q33. 6 Marks

A water tank has the shape of an inverted cone with height 12m and base radius 4m. Water is being pumped out at a rate of 2 m³/min. Find the rate at which the water level is falling when the water is 6m deep.

Answer:

Let h = height of water, r = radius of water surface
From similar triangles: r/h = 4/12 = 1/3, so r = h/3
Volume V = (1/3)πr²h = (1/3)π(h/3)²h = πh³/27
Given: dV/dt = -2 m³/min (negative because decreasing)
dV/dt = (dV/dh)(dh/dt) = (πh²/9)(dh/dt)
When h = 6m:
-2 = (π(36)/9)(dh/dt)
-2 = 4π(dh/dt)
dh/dt = -1/(2π) m/min
The water level is falling at 1/(2π) ≈ 0.159 m/min
