Chapter 3: Linear Equations in Two Variables

Introduction

Linear equations in two variables are algebraic expressions of the form ax + by + c = 0, where a, b, and c are real numbers, and a and b are not both zero. This chapter explores the properties, solutions, and graphical representations of these equations.

Definition

A linear equation in two variables is an equation that can be written in the form ax + by + c = 0, where a, b, and c are real numbers, and a and b are not both zero.

Important Points

  1. Every linear equation in two variables has infinitely many solutions.
  2. The graph of a linear equation in two variables is a straight line.
  3. If a point lies on the graph of a linear equation, it is a solution of the equation.
  4. The solution of a linear equation is not affected when:
    • The same number is added to (or subtracted from) both sides of the equation.
    • You multiply or divide both sides of the equation by the same non-zero number.
  5. An equation of the form y = mx represents a line passing through the origin.
  6. The equation of the x-axis is y = 0.
  7. The equation of the y-axis is x = 0.
  8. A linear equation in two variables represents a straight line on the Cartesian plane.
  9. The coordinates of every point on the line satisfy the equation of the line.
  10. Two lines in a plane can either be intersecting lines or parallel lines.
  11. If two lines intersect, they have exactly one point in common.
  12. If two lines are parallel, they have no point in common.
  13. If two lines are coincident, they have infinitely many points in common.
  14. The general form of a linear equation in two variables is ax + by + c = 0.
  15. The standard form of a linear equation in two variables is ax + by = c.
  16. A system of linear equations may have a unique solution, infinitely many solutions, or no solution.
  17. For the equations a₁x + b₁y + c₁ = 0 and a₂x + b₂y + c₂ = 0:
    • If a₁/a₂ ≠ b₁/b₂, the system has a unique solution.
    • If a₁/a₂ = b₁/b₂ = c₁/c₂, the system has infinitely many solutions.
    • If a₁/a₂ = b₁/b₂ ≠ c₁/c₂, the system has no solution.
  18. The graphical method can be used to solve a system of linear equations.
  19. The algebraic methods for solving systems of linear equations are:
    • Substitution method
    • Elimination method
    • Cross-multiplication method
  20. Linear equations can be used to solve real-life problems.

Examples

Example 1: Representing an equation graphically

Draw the graph of the equation x + y = 7.

Solution:

To draw the graph, we need at least two solutions of the equation.

When x = 0, y = 7 → (0, 7)

When y = 0, x = 7 → (7, 0)

Plot these points on the Cartesian plane and draw a line through them.

Example 2: Solving a system of equations

Solve the following system of equations:

2x + 3y = 12

x - y = 1

Solution:

Using substitution method:

From the second equation, x = y + 1

Substitute into the first equation: 2(y + 1) + 3y = 12

2y + 2 + 3y = 12 → 5y = 10 → y = 2

Then x = y + 1 = 2 + 1 = 3

So the solution is x = 3, y = 2.

Example 3: Word problem

The sum of the digits of a two-digit number is 9. If 27 is added to the number, the digits get reversed. Find the number.

Solution:

Let the units digit be x and tens digit be y.

Then the number is 10y + x

Given: x + y = 9 (Equation 1)

When 27 is added: 10y + x + 27 = 10x + y

Simplifying: 9y - 9x + 27 = 0 → x - y = 3 (Equation 2)

Adding Equations 1 and 2: 2x = 12 → x = 6

Then from Equation 1: y = 3

So the number is 10 × 3 + 6 = 36.

Quick Tips
